## 📖 Overview
The **Store Demo** project is a simple yet effective web application showcasing product management and CRUD operations.  
It demonstrates how to build a store interface with **Spring Boot, Java, and MySQL**, integrating backend and frontend for a smooth user experience.

---

## ✨ Features
- 📦 **Product Management** – Add, edit, delete, and list products  
- 🖼️ **Image Support** – Products can include images for better presentation  
- 🛠️ **CRUD Operations** – Full Create, Read, Update, Delete functionality  
- 📊 **Admin Interface** – Manage products and view store data

## 🚀 Getting Started

### Prerequisites
- Java 17  
- Maven 3.9+  
- MySQL 8.0+  

### Installation
```bash
# Clone the repository
git clone https://github.com/souhayl66/store-demo.git

# Navigate into the project
cd store-demo

# Build the project
mvn clean install

# Run the application
mvn spring-boot:run
